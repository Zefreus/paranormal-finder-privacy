Política de Privacidade

Paranormal Finder — Zefreus Labs
Última atualização: julho de 2026

Esta política descreve como o aplicativo Paranormal Finder, desenvolvido por Zefreus Labs, coleta, usa e armazena informações no seu dispositivo.

1. Dados coletados

O Paranormal Finder acessa os seguintes recursos do dispositivo exclusivamente para fornecer as funcionalidades do app:

  • Microfone: capturas de áudio são usadas em tempo real para análise de frequência (EVP) e Spirit Box. Gravações salvas pelo usuário ficam armazenadas localmente no dispositivo, na pasta Android/data/com.zefreus.paranormal/files/EVP/.
  • Câmera: utilizada pelo módulo SLS para detecção de formas via ML Kit. Nenhuma imagem é salva automaticamente.
  • Sensores de movimento e campo magnético: acelerômetro e magnetômetro são lidos em tempo real para os módulos EMF e Geofone. Os valores não são persistidos além da sessão ativa.
  • Localização: utilizada opcionalmente para registrar o local de uma sessão de investigação. O dado é salvo apenas localmente no dispositivo.

2. Armazenamento

Todos os dados gerados pelo app — gravações de áudio, logs de sessão e relatórios exportados — são armazenados exclusivamente no dispositivo do usuário. O app não possui servidores próprios e não transmite nenhum dado para a internet.

3. Compartilhamento de dados

O Paranormal Finder não compartilha, vende ou transfere dados do usuário para terceiros. A única forma de compartilhamento é quando o próprio usuário usa a função "Exportar Relatório", que abre o compartilhamento nativo do Android para que o usuário escolha o destino.

4. Serviços de terceiros

O app utiliza as seguintes bibliotecas de terceiros, sujeitas às suas próprias políticas de privacidade:

  • Google ML Kit (Pose Detection): processamento de imagem realizado inteiramente no dispositivo, sem envio de dados ao Google. Termos do ML Kit

5. Permissões utilizadas

  • CAMERA — detecção de formas humanoides (módulo SLS)
  • RECORD_AUDIO — análise de áudio (módulos EVP e Spirit Box)
  • ACCESS_FINE_LOCATION — registro opcional do local da sessão
  • HIGH_SAMPLING_RATE_SENSORS — leitura de alta frequência do acelerômetro para detecção de microvibrações (módulo Geofone)
  • USB_HOST — comunicação com câmera térmica externa via USB-C

Nenhuma permissão é usada para fins publicitários ou de rastreamento.

6. Dados de menores

O Paranormal Finder não é direcionado a menores de 13 anos e não coleta intencionalmente dados de crianças.

7. Alterações nesta política

Eventuais atualizações serão publicadas nesta página. O uso continuado do app após alterações constitui aceite da política revisada.
